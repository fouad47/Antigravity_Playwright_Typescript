# Chapter 04: Element Interactions

## 🎯 Learning Objectives
- Master all element interaction methods
- Understand click, fill, check, select, hover, drag
- Learn keyboard and mouse actions
- Handle file uploads

## 📁 Files
| File | Description |
|------|-------------|
| `click-and-fill.spec.ts` | Click, fill, type, clear interactions |
| `checkbox-dropdown.spec.ts` | Check, uncheck, selectOption |
| `hover-dragdrop.spec.ts` | Hover, drag and drop, mouse actions |
| `keyboard-actions.spec.ts` | Keyboard interactions, shortcuts |
| `file-upload.spec.ts` | File upload interactions |

## 🏋️ Mini Challenges
1. Fill a form and submit using keyboard (Enter key)
2. Select all dropdown options one by one
3. Implement drag and drop using mouse events
4. Upload a dynamically created file and verify

## 🎤 Interview Questions
1. What is the difference between fill() and type()?
2. How does Playwright handle file uploads?
3. How do you simulate keyboard shortcuts?
4. What is the difference between click() and dblclick()?
